# Chrome 网站收藏管理助手 PRD

## 1. 产品概述

### 1.1 产品名称
Chrome 网站收藏管理助手

### 1.2 产品定位
一款用于管理“低频但有保留价值”网站的 Chrome 收藏插件。
帮助用户把不适合长期放在书签栏的网站统一归档，并通过卡片化方式进行查看、搜索、排序和同步管理。

### 1.3 产品目标
- 降低 Chrome 书签栏拥挤度
- 提供更高效的收藏分类和整理方式
- 提供比原生书签更直观的新标签页浏览体验
- 支持导入导出及多端同步，保证数据可迁移和可延续

### 1.4 目标用户
- 经常浏览大量网站、但只有部分网站高频使用的用户
- 有分类整理需求的办公、学习、内容收集类用户
- 希望在新标签页快速查看收藏内容的用户

## 2. 使用场景

### 2.1 场景一：快速收藏当前网页
用户浏览网页时，发现页面未来可能还会用到，但又不想加入顶部书签栏。
此时点击插件图标，选择分类，即可完成收藏。

### 2.2 场景二：在新标签页浏览和管理收藏
用户打开新标签页，希望像看卡片墙一样查看各类收藏网站，并进行排序、删除、搜索和整理。

### 2.3 场景三：批量收起当前工作现场
用户打开了一批临时网页，希望一键保存当前窗口所有标签页，归档到临时分类中，并关闭这些标签页。

### 2.4 场景四：数据迁移与恢复
用户希望导入 Chrome 原有书签，或将本插件数据导出备份，避免数据丢失。

## 3. 产品范围

### 3.1 本期范围
- 当前网页快速收藏
- 新标签页卡片化展示
- 分类管理与拖拽排序
- 卡片分类内排序
- 卡片跨分类拖拽
- 支持将右侧卡片直接拖到左侧分类
- 搜索功能
- 导入导出 Chrome 书签文件
- Chrome 同步能力接入
- 一键保存当前全部标签页到临时分类并关闭
- 深色 / 浅色模式
- 左侧分类栏收起 / 展开

### 3.2 暂不包含
- 收藏标签系统
- 团队共享收藏
- 云端独立账号体系
- 链接有效性检测

## 4. 功能需求

### 4.1 功能一：快速收藏当前网页

#### 4.1.1 功能说明
用户点击浏览器工具栏插件图标，弹出收藏面板，展示分类列表。
用户点击某个分类后，将当前网页保存到该分类。

#### 4.1.2 交互流程
1. 用户打开任意网页
2. 点击插件图标
3. 弹窗展示已有分类
4. 用户点击目标分类
5. 系统保存当前页面信息
6. 提示收藏成功

#### 4.1.3 保存字段
- 页面标题
- 页面 URL
- favicon
- 所属分类

#### 4.1.4 补充要求
- 支持在弹窗内新建分类
- 同一分类下若 URL 已存在，提示“已收藏”，不重复创建
- 收藏成功后支持轻提示反馈

#### 4.1.5 异常处理
- 特殊页面如 `chrome://`、空白页、插件页不可收藏
- favicon 获取失败时使用默认占位图标

### 4.2 功能二：新标签页卡片式管理页

#### 4.2.1 功能说明
新打开 Tab 页时，展示收藏管理页面。

#### 4.2.2 页面布局
- 左侧：分类列表
- 右侧：收藏卡片区域
- 左侧支持展开 / 折叠
- 右侧跟随左侧状态自适应布局

#### 4.2.3 卡片内容
- 网站 favicon
- 网站标题
- 网站域名或简化 URL

#### 4.2.4 展示方式
- 按分类分组展示
- 分类顺序与左侧列表一致
- 每个分类下展示对应收藏卡片

#### 4.2.5 卡片操作
- 点击卡片打开网站
- 支持删除卡片
- 支持拖动排序
- 支持跨分类移动

#### 4.2.6 空状态
- 无分类时：提示创建分类
- 无收藏时：提示添加网站
- 搜索无结果时：提示未找到匹配内容

### 4.3 功能三：分类管理

#### 4.3.1 功能说明
用户可对分类进行新增、重命名、删除和排序。

#### 4.3.2 支持操作
- 新增分类
- 重命名分类
- 删除分类
- 左侧分类拖动排序

#### 4.3.3 联动规则
- 左侧分类顺序变化后，右侧分类展示顺序同步变化

#### 4.3.4 删除规则
删除分类时，需弹出确认框，并支持以下策略之一：
- 删除分类及其下所有收藏
- 将该分类下收藏移动到其他分类

本期建议默认：
- 删除分类及其下收藏
- 后续版本再支持移动

### 4.4 功能四：卡片拖拽排序与跨分类拖拽

#### 4.4.1 功能说明
用户可在右侧内容区域内，对卡片进行同分类排序和跨分类移动。

#### 4.4.2 规则
- 支持同一分类内拖拽排序
- 支持从右侧一个分类拖到右侧另一个分类
- 支持将右侧卡片直接拖到左侧分类列表中的某个分类项
- 当卡片拖入目标分类后，自动追加到该分类的最后一个位置
- 卡片拖动后需立即更新所属分类和排序字段
- 排序结果本地保存
- 刷新页面后顺序保持一致

#### 4.4.3 交互要求
- 左侧分类项在可接收卡片拖入时，需要有明确高亮反馈
- 卡片拖入左侧分类并释放后，右侧内容区域同步更新
- 若拖拽失败或落点无效，则卡片回到原位置

### 4.5 功能五：搜索功能

#### 4.5.1 功能说明
支持对收藏网站进行搜索，帮助用户快速定位目标网站。

#### 4.5.2 搜索范围
- 网站标题
- URL

#### 4.5.3 交互要求
- 页面顶部提供搜索框
- 输入关键词后实时显示结果
- 支持模糊匹配
- 清空关键词后恢复原始展示
- 搜索结果保持原分类分组展示

### 4.6 功能六：导入 / 导出

#### 4.6.1 功能说明
支持导入和导出 Chrome 书签格式文件。

#### 4.6.2 导入要求
- 支持导入 Chrome 导出的书签文件
- 将书签文件中的文件夹转换为插件分类
- 将书签项转换为收藏卡片

#### 4.6.3 导出要求
- 支持将插件收藏数据导出为 Chrome 兼容格式
- 导出时保留分类结构

#### 4.6.4 重复处理
导入时若发现重复 URL：
- 默认跳过重复项
- 导入完成后展示统计结果

#### 4.6.5 导入结果反馈
- 导入分类数
- 导入网址数
- 重复跳过数
- 失败数

### 4.7 功能七：Chrome 同步

#### 4.7.1 功能说明
通过 Chrome 浏览器同步机制，将用户数据同步到同账号下的其他设备。

#### 4.7.2 同步内容
- 分类数据
- 收藏数据
- 分类顺序
- 卡片顺序
- 主题设置
- 左侧栏收起状态

#### 4.7.3 实现要求
- 优先使用 `chrome.storage.sync`
- 本地使用 `chrome.storage.local` 做兜底
- 若同步容量不足，提示用户部分数据仅保存在本地

### 4.8 功能八：一键保存当前全部打开标签页

#### 4.8.1 功能说明
支持将当前窗口中已打开的网页一键保存到临时分类，并关闭这些标签页。

#### 4.8.2 操作流程
1. 用户点击“一键保存当前全部标签页”
2. 系统弹出确认框
3. 生成临时分类
4. 将当前窗口全部可保存页面加入该分类
5. 关闭已保存标签页
6. 提示处理结果

#### 4.8.3 临时分类命名
默认格式：
`临时收藏-YYYY-MM-DD HH:mm`

#### 4.8.4 过滤规则
以下页面默认跳过：
- `chrome://`
- `edge://`
- 插件页
- 空白页
- 无法访问的系统页

#### 4.8.5 结果反馈
- 成功保存数
- 跳过数
- 关闭数

### 4.9 功能九：主题模式

#### 4.9.1 功能说明
支持浅色模式和深色模式。

#### 4.9.2 模式选项
- 浅色模式
- 深色模式
- 跟随系统

#### 4.9.3 要求
- 用户切换后立即生效
- 模式设置需持久化保存
- 支持同步到其他设备

## 7. 权限与技术约束

### 7.1 可能需要的 Chrome 权限
- `storage`
- `tabs`
- `bookmarks`
- `favicon`
- `activeTab`

若新标签页替换默认页，需配置对应扩展能力。

### 7.2 技术注意点
- `chrome.storage.sync` 有容量限制，需要控制数据体积
- favicon 获取需考虑失败场景
- 导入导出 Chrome 书签格式时需保证兼容性
- 批量关闭标签页前必须确认，避免误操作

## 8. 非功能需求

### 8.1 性能
- 插件弹窗打开后应快速展示分类列表
- 收藏操作应在 1 秒内完成
- 新标签页首屏渲染流畅
- 搜索结果应实时反馈

### 8.2 可用性
- 拖拽反馈清晰
- 删除和批量关闭需有确认提示
- 无数据、无结果、异常情况需有明确提示

### 8.3 兼容性
- 支持主流 Chrome 桌面版
- 适配常见宽屏和笔记本分辨率

## 9. 验收标准

### 9.1 快速收藏
- 能从插件弹窗中选择分类并成功保存当前网页
- 同分类重复 URL 不重复创建

### 9.2 卡片页
- 新标签页正确展示分类及卡片
- 左侧可收起 / 展开
- 右侧按分类显示卡片

### 9.3 排序与拖拽
- 左侧分类支持拖拽排序
- 右侧分类顺序同步变化
- 卡片支持分类内拖拽排序并持久化
- 卡片支持跨分类拖拽并正确更新所属分类
- 卡片支持直接拖到左侧分类项，并自动落到目标分类最后一个位置

### 9.4 搜索
- 可根据标题和 URL 搜索
- 搜索结果正确更新

### 9.5 导入导出
- 能成功导入 Chrome 书签文件
- 能导出兼容 Chrome 的书签文件

### 9.6 批量保存标签页
- 能将当前窗口全部可保存标签页加入临时分类
- 已保存页面可被关闭
- 特殊页面可被正确跳过

### 9.7 主题与同步
- 支持浅色 / 深色切换
- 设置和收藏内容可通过 Chrome 账号同步
