# Chrome Web Store Listing

## Title

OneMoreMark - New Tab Bookmark Manager

## Short description

Turn your new tab into a clean visual bookmark dashboard with categories, cards, import/export, and Chrome sync.

## Detailed description

OneMoreMark turns your Chrome new tab into a clean visual bookmark dashboard.

Instead of keeping important sites in a crowded bookmark bar or scattered folders, OneMoreMark lets you organize them as simple cards grouped by category. Every new tab becomes a calm starting point for the websites you actually use.

Key features:
- Replace the default new tab page with a visual bookmark dashboard
- Organize websites into categories and card-style layouts
- Save the current page or all open tabs from the popup
- Drag bookmarks between categories to reorganize quickly
- Search bookmarks by name or URL
- Import and export bookmarks as an HTML file
- Sync your setup with Chrome storage when available
- Switch between light mode, dark mode, and browser-matched appearance
- Use English or Simplified Chinese interface language

OneMoreMark is designed for people who open the same websites every day, keep too many bookmarks in the browser bar, or want a cleaner personal start page.

Your data stays in your browser storage. You can export your bookmarks anytime for backup or migration.
